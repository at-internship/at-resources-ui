# Built & Run
- $ npm install
- $ npm-check-updates -D

- $ npm run dev

**Unit Test**
- $ npm install mocha --save-dev
- $ npm install chai --save-dev
- $ npm run test

**Code Coverage**
- $ npm install nock --save-dev
- $ npm run coverage
